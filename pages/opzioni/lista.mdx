# Opzione Lista

Le opzioni di tipo lista in Welcapp sono un tipo di opzione utilizzato per organizzare altre opzioni in una lista.

Le opzioni di tipo lista possono contenere altre opzioni di tipo base, di tipo posizione o di altri tipi, e possono essere utilizzate per rappresentare collezioni di dati, ad esempio una lista di prodotti o di servizi disponibili.

## Dati

- nome
- immagine
- descrizione
- opzioni

## Layout

Il layout di un'opzione di tipo lista mostra in alto il nome e la descrizione dell'opzione,
a seguire mostra tutte le sotto opzioni come elementi di una lista.
