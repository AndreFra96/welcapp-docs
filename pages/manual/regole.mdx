# Opzione Regole

L'opzione di tipo regole è un'opzione specializzata utilizzata in Welcapp per comunicare agli ospiti le regole dell'appartamento; Gli host possono utilizzare questa opzione per fornire agli ospiti una lista completa delle regole dell'appartamento in modo chiaro e facilmente accessibile.

L'opzione di tipo regole è particolarmente utile per fornire agli ospiti le informazioni di cui hanno bisogno per rispettare le regole dell'appartamento e assicurare un soggiorno piacevole per tutti.

## Dati

- nome
- immagine
- descrizione
- regole
